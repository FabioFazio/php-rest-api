# Basic PHP API Rest

This project has been generated as a spike from [Corey Maynard tutorial](http://coreymaynard.com/blog/creating-a-restful-api-with-php)

Our goals:
  - Keep code simple as possibile and easy to use
  - Keep previous versions of API to support full compatibility
  - Move configuration to a separate file
  - Generate an automthic documentation of each Entity/Service
  - Support a log services
  - Implement a valid solution to prevent CSRF attacks
